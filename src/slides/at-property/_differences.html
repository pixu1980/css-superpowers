What are the differences between registered and unregistered custom properties?

Registered properties are animatable and transitionable

    
<button class="unregistered">Background Not Registered</button>
<button class="registered">Background Registered</button>
    
  
    
button.unregistered {
  --unregistered--coffee-color: #c0ffee;
  background-color: var(--unregistered--coffee-color);
  transition: --unregistered--coffee-color 1s ease-in-out; // this will NOT work

  &:hover,  &:focus {
    --unregistered--coffee-color: #b4d455;
  }
}

button.registered {
  --coffee-color: #c0ffee;
  background-color: var(--coffee-color);
  transition: --coffee-color 1s ease-in-out; // this will work

  &:hover, &:focus {
    --coffee-color: #b4d455;
  }
}
