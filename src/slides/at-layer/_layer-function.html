What about old CSS codebase and third party dependencies?

The layer() function will be your best friend

@import ... layer() pattern

    
@import 'path-to-old-dependency.css' layer(old-library.old-dependency);
@import 'path-to-old-dependency.theme.css' layer(old-library.old-dependency);
@import 'path-to-other-old-dependency.css' layer(old-library.other-old-dependency);
      
@layer new-library {
  ...
}
