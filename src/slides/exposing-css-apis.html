Building CSS APIs

combining variables and layers

Let's assume we want to create a Design System

        
@layer reset {
  * {
    box-sizing: border-box;
    ...
  }
}
  
@layer components {
  button {
    border-size: 1px solid lightgray;
  }
}
  
@layer layout {
  body {
    display: grid;
    grid-template-areas:
      'header header'
      'aside main'
      'footer footer';
  }
}

"Esecuzione" dei layer nell'ordine che si desidera

        
@layer reset, layout, components;

Livelli di Profondità e Override (Reverse Cascade)

        
:root {
  --ui--font-size: 10px;
}
        
@layer ui {
  :root {
    --ui--font-size: 16px;
  }
}
  
@layer ui.button {
  :root {
    --ui--font-size: 18px;
  }
  
  button {
    font-size: var(--ui--font-size);
  }
}
  
@layer ui.input {
  :root {
   --ui--font-size: 20px;
  }
  
  input {
    font-size: var(--ui--font-size);
  }
}
